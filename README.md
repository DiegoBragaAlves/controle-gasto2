# Controle de Gastos (BootCamp II)

## Descrição
Aplicação CLI em Python para controle de gastos pessoais.

## Funcionalidades
- Adicionar gastos
- Listar gastos
- Remover gastos
- Ver total de gastos
- Consultar cotação do dólar (API)

## API utilizada
- AwesomeAPI

## Como executar

pip install requests  
python main.py

## Teste

python test_api.py
